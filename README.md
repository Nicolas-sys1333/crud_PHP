<h1>✏️ CRUD</h1>
Criação de um formulário para cadastro de usuários com as funcionalidades do CRUD: Criar, Ler, Atualizar e Excluir

- Nome Completo
- Data de Nascimento
- Cidade
- Email
- Senha
